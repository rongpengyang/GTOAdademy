# QA Build Test Review
After each feature:
1. Build the app.
2. Run unit tests.
3. Inspect warnings.
4. Check SwiftUI previews if available.
5. Review for architecture drift.
6. Review for App Store compliance.
7. Report:
   - Passed checks
   - Failed checks
   - Files changed
   - Bugs fixed
   - Remaining risks
