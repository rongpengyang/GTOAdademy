# Poker Strategy Domain
Default game:
- 6-max No-Limit Texas Hold'em cash game
- 100bb effective stack
- Educational simulator only, no real-money gaming
Content rules:
- Explain poker concepts clearly for beginners.
- Always distinguish exploitative advice from GTO-inspired advice.
- Do not claim perfect GTO unless using verified solver data.
- Every quiz must include correct answer, incorrect-option explanations, and learning objective.
- Use configurable thresholds for VPIP/PFR/player type classification.
- Treat ranges as training approximations, not universal truth.
