# App Store Compliance
Check:
- No real-money gambling.
- No prizes, cash-out, lottery, sweepstakes, or casino links.
- No purchasable chips or currencies connected to real-money gaming.
- Paid download is handled by App Store pricing, not inside the app.
- Any future digital feature unlock must use Apple IAP.
- No dynamic executable code download.
- Privacy policy must disclose data collection and any third-party AI sharing.
- Metadata must accurately describe the app.
- App Review Notes must clearly explain this is a poker education/training simulator.
